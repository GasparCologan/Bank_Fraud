def asignar_cuatrimestre(df, mes_columna):
    cuatrimestre_values = {
        0: 1,
        1: 1,
        2: 1,
        3: 2,
        4: 2,
        5: 2,
        6: 3,
        7: 3,
        8: 3,
        9: 4,
        10: 4,
        11: 4
    }

    if mes_columna not in df.columns:
        print(f"La columna '{mes_columna}' no se encuentra en el DataFrame.")
        return None

    df['Quarter'] = df[mes_columna].map(cuatrimestre_values)
    return df

Undersampling:¶

Para reducir tiempos de computacin a la hora de ejecutar un problema con un milln de instancias, y para elegir el modelo a utilizar, realizamos undersampling. Ahora el dataset se nos queda en un 60% de entrenamiento que estar rebalanceado, y una validacin y test que permanecen vrgenes. Siempre utilizamos el mismo random_state para los splits
